# 新增昏厥
你可以在配置文件进行昏厥的调控，决定是固定的昏厥时长还是随机范围的昏厥时长，范围你当然可以自定义
# 增加打胶与help
可以通过ccbhelp来查看可用命令\
打胶无需多言
## 增加了一条伪命令（仅管理员可用） 
因为发现在测试的时候由于意外导致的cd非常难熬\
而增加一条命令工作量过大\
于是产生了这条伪命令通过检测词条完成执行\
/ccb clear 清理计时器,如果后方带有@的人物则清理他人的计时器
